My Projects

AstroWatch

A comprehensive Satellite Tracking and Collision Detection System, designed to provide dynamic insights into Earth-orbiting satellites. It accurately visualizes satellites in a 3D Earth orbit, offers a detailed 2D ground track for individual satellites, and crucially, performs real-time collision risk assessments by retrieving the latest satellite data from Celestrak. The system is built using Python, with key libraries including PyQt5 for the graphical user interface, Skyfield for precise orbital calculations, Matplotlib for data visualization, and NumPy along with SciPy for robust numerical computations.

Python PyQt5 Matplotlib Skyfield QSS API Integration

Bookstore Application System

A comprehensive Bookstore Application developed to manage book inventory and customer interactions within a single-window graphical user interface (GUI). Built using JavaFX and NetBeans IDE, this project focuses on a robust design incorporating the State Design Pattern. The application distinguishes between two user types: an Owner and Customers. The Owner has administrative capabilities, including adding and deleting books and registering new customer accounts. Customers can browse available books, make purchases , and utilize a loyalty point system where points can be accumulated and redeemed against transaction costs.
